# data-ai-training
Personal repository for learning and experimenting with data engineering and AI concepts.

## Repository Structure
- `data-engineering/`: ETL pipelines, data warehousing, orchestration
- `machine-learning/`: ML algorithms, deep learning, model deployment
- `mlops/`: MLOps tools and practices
- `big-data/`: Spark, Hadoop, distributed computing
- `cloud-platforms/`: AWS, Azure, GCP implementations

## Tech Stack
- Python 3.x
- Libraries: pandas, scikit-learn, TensorFlow, PyTorch, PySpark
- Tools: Docker, Airflow, MLflow, dbt
